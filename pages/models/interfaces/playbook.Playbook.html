globalID

globalID: string
  • Defined in playbook/playbook.ts:25

Global identifier for the playbook. This is set by the original creator of the playbook and will remain the same even if the playbook is bundled into a kit and installed on another system.

Each user may only have one playbook with a given global ID, but multiple users could each have a copy of a playbook with the same global ID.
